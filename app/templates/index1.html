About the System

Project Overview

The Chronic Disease Prediction System is an advanced machine learning application designed to predict the likelihood of five major chronic diseases: Heart Disease, Diabetes, Kidney Disease, Asthma, and Hypertension.

Our system uses a comprehensive approach combining demographic data, clinical measurements, lifestyle factors, and symptom analysis to provide accurate predictions with 95.67% accuracy.

Methodology

  • Data Collection: Synthetic but realistic dataset with 1,500 records (300 per disease)
  • Feature Engineering: TF-IDF vectorization for text features, normalization for numerical features
  • Model Selection: Tested Logistic Regression, Random Forest, and SVM algorithms
  • Best Model: Logistic Regression with 95.67% accuracy
