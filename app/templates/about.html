About the System

Chronic Disease Prediction System

This advanced machine learning system predicts the likelihood of chronic diseases based on patient demographics, clinical measurements, lifestyle factors, and symptoms.

Diseases Predicted:
  • Heart Disease
  • Diabetes
  • Asthma
  • Hypertension
  • Kidney Disease
Technology Stack:
  • Machine Learning: Logistic Regression (95.67% accuracy)
  • Text Processing: TF-IDF Vectorization
  • Data Processing: Scikit-learn, Pandas, NumPy
  • Web Framework: Flask with Bootstrap UI
  • Dataset: 1,500 synthetic patient records
